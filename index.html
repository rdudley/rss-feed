Simple Yahoo RSS Feed Reader

This is a very simple RSS Feed reader that will:

  1. Display the top 5 "Latest News and Headlines" items from Yahoo's Latest News & Headlines RSS Feed
  2. Provide for a simple search within the same feed displaying the top 5 results
  3. Cache feed contents and check against the feed TTL if working directory permissions allow
  4. Check for invalid dates in the RSS feed and format item dates to a default timezone

This example uses asynchronous client-side requests to fetch data from the server. The server-side scripts are arranged in a very simplistic manner to show an understanding of MVC and OOP design patterns, but the example has kept functionality of these scripts limited and to work within the project specifications. A simple solution for a simple task.

Source files are available for download in zip or tar formats, and available at Github.
